libCyl is the best cylinder drawing library for swing


Well, Kyle Charters has done it again! Recently, the developer of the bukkit plugin Prefixed and main developer of the new upcoming game "Ascendance" has taken a leap and created a new, must-have program for any Java developer. The program is called, LibCyl. This amazing new piece of technology draws cylinders of specified height, depth and width, and allows you to customize the side and top colour of your newly generated cylinder. Clearly this advancement will revolutionize the way games and even the very base of the Java programming language will work. With the power of LibCyl, any of our cylinder-making needs can be fulfilled, as Kyle Charters says himself in the new programs slogan: "SIMPLE. ELEGANT. PERFECT." (Charters, LibCyl). Clearly a groundbreaking advancement in programming, we look forward to a newer, cleaner and cylinder-filled future.
